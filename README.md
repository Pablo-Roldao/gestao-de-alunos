# Sistema simplificado para gestão de alunos

Um sistema armazena dados de alunos, utilizando <strong>alocação dinâmica de memória e arquivos</strong>. O sistema é capaz de armazenar, inicialmente, 10 registros de alunos e aumentar a capacidade em 10 novos registros sempre que necessário.
